Requisistos | TermoCrivo

O sistema deve cadastrar paciente
O sistema deve importar imagem
O sistema deve fazer o processamento da imagem
O sistema deve gerar relatório a partir da imagem, transformando os valores de cor obtidos em porcentagens e se possível em temperatura.
O sistema deve criar gráficos com os valores de cor baseado nas porcentagens dos relatórios
O sistema deve mostrar lista de todos os pacientes com seus respectivos arquivos de relatórios e imagens
O sistema deve pesquisar na lista pacientes pelos dados específicos
